# Melodic Dictation

Harmonic melodic dictation page.

Features:
- Fixed 2-bar melodic dictation
- Melodies imply a simple harmonic progression such as I-V, I-IV, ii-V, i-V, or iv-V
- Chord tones appear on strong beats; passing/neighbor notes appear on weaker positions
- C major, G major, F major, A minor
- Stepwise, up to thirds, or mixed small leaps
- Rhythm includes quarters, eighths, dotted rhythms, dotted quarters, and triplets
- Three answer choices with notation
- Correct and selected notation in PDF export
- Real piano playback through Tone.js sampler
