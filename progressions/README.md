# Chord Progressions

Rebuilt from zero.

Features:
- Three-choice chord progression ear training
- Major/minor, triads/sevenths
- White-key keys or all keys
- Internal synth playback with count-in
- Four-part-style voicing
- Grand-staff notation
- Key signature shown in every notation example
- Theory spelling based on key, scale degree, chord quality, and chord tone
- Flat keys stay flat; sharp-only MIDI spelling is not used for notation
- PDF export includes correct and selected notation
